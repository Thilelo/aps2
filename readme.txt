Atividade de controle de versão - Git

Olá, pessoal

Na aula passada falamos sobre gerência de configuração e mudança, dentro do qual falamos sobre Git, e criamos nosso primeiro repositório usando esta ferramenta de versionamento de código.

Nesta atividade, iremos aprofundar os conhecimentos sobre Git. Para isso, realize os passos a seguir:

1) Assistir todos os vídeos dessa playlist sobre Git:
- https://youtube.com/playlist?list=PLInBAd9OZCzzHBJjLFZzRl6DgUmOeG3H0

2) Ao assistir as aulas, exercitar o Git a partir da criação de um repositório público seu (pode ser em qualquer plataforma git gratuita disponível, tais como Gitlab ou Github, entre outras), realizando os mesmos passos apresentados nos vídeos. 

3) Como entrega desta atividade, você deve enviar o link do seu repositório.
